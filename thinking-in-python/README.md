# 四 python-spider(网络爬虫小项目)

## 1、环境

python3

## 2、目录

1)数据存储，/u01/spider/data/

2)系统日志，/u01/spider/logs/spider.log

## 3、启动 

nohup python3.7 -u manage.py > /u01/spider/logs/spider.log &

根据python版本将python3.7改变为对应版本。
